# J8-Series-dataset 东南大学 J8 教学楼行人数据集 (俯视视角)

## 简介
本数据集包含在东南大学 J8 教学楼内拍摄的行人图像，采用**俯视视角**拍摄。数据集分为两个子集：
- **J8-1**：标注目标为**行人头部**（适用于头部检测、目标计数等任务）。
- **J8-2**：标注目标为**行人全身**（适用于行人检测、目标计数等任务）。

所有原始视频以 `.mp4` 格式存储，对应的标注文件为 `.csv` 格式，包含了每帧中目标的边界框信息。

## 标注格式 (CSV)
每个 `.csv` 文件与同名的 `.mp4` 视频文件对应，包含视频中每一帧的目标标注信息。CSV 文件的具体列定义如下（示例，请按实际情况调整）：

| 列名 | 类型 | 说明 |
|------|------|------|
| `frame(第一列)` | int | 帧序号 |
| `ID(第二列)` | int | 对象ID |
| `x` | int | 边界框左上角 x 坐标 (像素) |
| `y` | int | 边界框左上角 y 坐标 (像素) |
| `w` | int | 检测框宽度 (像素) |
| `h` | int | 检测框高度 (像素) |

*注：后续的-1为占位符。*

## 采集与标注
- **拍摄地点**：东南大学 J8 教学楼入口区域。
- **拍摄视角**：俯视，无人机拍摄。
- **拍摄时间**：包括课前、课后不同时段，涵盖不同光照条件。
- **标注工具**：[CVAT,Darklabel]。
